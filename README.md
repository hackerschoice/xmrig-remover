# xmrig-remover
Detects and kills XMRig processes under Linux


```shell
bash -c "$(curl -fsSL https://thc.org/xmrig)"
```
or
```shell
bash -c "$(curl -SsfL https://github.com/hackerschoice/xmrig-remover/raw/refs/heads/main/xmrig-remover.sh)"
```
